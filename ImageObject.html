The ImageObject type allows you to provide details about an image file (in any format) such as its caption, thumbnail and further information inherited from MediaObject.

Examples

A picture of a kiwi (the bird, not the people).

{
  "type": "ImageObject",
  "caption": "Kiwi",
  "contentSize": "10.4",
  "contentUrl": "http://www.example.com/kiwi.png",
  "encodingFormat": "image/png",
  "thumbnail": {
    "type": "ImageObject",
    "contentUrl": "http://www.example.com/kiwi_mini.png"
  }
}

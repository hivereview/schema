Examples

To illustrate how Heading nodes are represented in alternative formats, we'll use the following example.

{
  "type": "Heading",
  "depth": 2,
  "content": ["Secondary Heading"]
}

Encodings

This section describes common encodings for this node type. These samples are generated from the above examples by Encoda, but you can also author them in each format.

HTML

HTML supports Heading nodes with the <h1> to <h6> elements.

<h2 id="secondary-heading">Secondary Heading</h2>

JATS

JATS lacks a depth attribute so this is lost on conversion, the <title> element is used.

<sec>
  <title>Secondary Heading</title>
</sec>

YAML

type: Heading
depth: 2
content:
  - Secondary Heading

Markdown

Markdown headings are denoted by a number of hashes (#) equal to the depth.

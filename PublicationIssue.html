A part of a successively published publication such as a periodical or publication volume, often numbered, usually containing a grouping of works such as articles. In reproducible documents, Articles can reference a PublicationIssue for bibliographical/citation purposes.

Examples

An example of an issue of the journal Nature is below. Note that the issue has an issueNumber and isPartOf a volume that, in turn, isPartOf the Periodical with the title "Nature".

{
  "type": "PublicationIssue",
  "url": "https://www.nature.com/nature/volumes/571/issues/7766",
  "issueNumber": 571,
  "datePublished": "2019-07-25",
  "isPartOf": {
    "type": "PublicationVolume",
    "volumeNumber": 571,
    "isPartOf": {
      "type": "Periodical",
      "title": "Nature"
    }
  }
}
